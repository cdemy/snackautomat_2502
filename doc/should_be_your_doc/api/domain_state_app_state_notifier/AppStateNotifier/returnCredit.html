void returnCredit()

Returns all current credit to the user.

Refunds the entire credit balance back to the user and resets the credit to 0. Also clears any currently selected snack as part of the return process.

If there is no credit to return (balance is 0), an appropriate message is displayed. Otherwise, the returned amount is shown in euros with two decimal places.

This function simulates the "coin return" button found on physical vending machines.

Implementation

void returnCredit() {
  if (state.currentCredit > 0) {
    final returnedAmountInEuro = state.currentCredit / 100.0;
    state = state.copyWith(currentCredit: 0);
    print(
      'Guthaben zurückgegeben: ${returnedAmountInEuro.toStringAsFixed(2)} €',
    );
    print('Neues Guthaben: ${state.currentCredit / 100} €');
  } else {
    print('Kein Guthaben zum Zurückgeben vorhanden.');
  }
  state = state.copyWith(selectedSnackId: null);
}
