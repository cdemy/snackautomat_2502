AppStateNotifier()

Creates a new AppStateNotifier with initial vending machine state.

Initializes the vending machine with a predefined set of snacks:

  • Schokolade (Chocolate): 1.50€, quantity 10
  • Chips: 1.20€, quantity 5
  • Cola: 2.00€, quantity 8

The initial credit is set to 0 and no snack is pre-selected.

Implementation

          Snack(id: '2', name: 'Chips', price: 120, quantity: 5),
          Snack(id: '3', name: 'Cola', price: 200, quantity: 8),
        ],
      ),
    );
