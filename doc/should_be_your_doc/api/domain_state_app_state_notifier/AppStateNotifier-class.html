AppStateNotifier

AppStateNotifier class

A state notifier that manages the vending machine's application state.

This class extends StateNotifier to provide reactive state management for the snack vending machine application. It handles operations such as adding credit, selecting snacks, purchasing items, and managing the overall state of the vending machine.

The state includes available snacks, current credit balance, and the currently selected snack ID. All monetary values are stored internally as cents (integers) to avoid floating-point precision issues.
